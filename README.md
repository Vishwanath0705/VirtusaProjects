# Multi-Project Repository: Python, SQL & Java Systems

This repo contains three complete projects across different technologies—Python, SQL, and Java—designed to demonstrate practical problem-solving, clean architecture, and real-world application development.

Projects Included

## Resume Analyzer & Job Matcher (Python + Streamlit) Overview
A smart resume analysis application that compares a resume with a job description and generates an ATS score with detailed improvement suggestions. It also includes an optional AI-powered enhancement system using Google Gemini AI.

Features Add resume (PDF/DOCX) Input job description ATS score calculation Skill gap detection Experience and project evaluation Resume improvement suggestions Score visualization AI-based analysis (optional) Tech Stack Python Streamlit Matplotlib PyPDF2 / python-docx Google Gemini AI (optional)

-> How to Run streamlit run app.py

-> Key Concepts Used ATS scoring system NLP-based text processing File handling (PDF/DOCX) Data visualization AI integration

## Hospital Management System (SQL Analytics Project) Overview
A relational database project designed for hospital data management and generating meaningful insights using SQL queries.

Features Patient, doctor, appointment, and treatment management Revenue analysis Doctor performance tracking Disease frequency analysis Patient visit analysis Tech Stack MySQL SQL Joins (INNER, LEFT, GROUP BY, HAVING) Aggregate Functions

-> How to Run Create database Run schema.sql Run data.sql Run analysis.sql

-> Key Concepts Used Joins and aggregations Group by analysis Data relationships using foreign keys Business analytics using SQL

## Quiz Application (Java Swing + MySQL) Overview
A desktop-based quiz application with admin and student modules. Admin manages questions while students can register, take quizzes, and view results.

Features Admin: add, update, delete questions Student registration and login Quiz system with scoring Result generation and display Database integration Tech Stack Java Swing MySQL JDBC

-> How to Run javac Main.java java Main

-> Key Concepts Used Object-Oriented Programming (OOP) Event-driven programming Database connectivity GUI development
