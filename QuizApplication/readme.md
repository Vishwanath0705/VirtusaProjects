# Quiz Application (Java Swing + MySQL)

## Overview
This is a desktop-based Quiz Application developed using Java Swing and MySQL. It supports both Admin and Student modules with full database integration.

Admin can manage questions and view results, while students can register, login, take quizzes, and view results.

---

## Features

### Admin Panel
- Add new questions
- Update existing questions
- Delete questions
- View all questions
- View student results
- Admin menu dashboard

### Student Panel
- Enter student details
- View instructions before quiz
- Take quiz
- Submit answers
- View result

### Authentication
- User registration
- Login system
- Forgot password recovery

---

## How to Run

1. Install Java (JDK 8+)
2. Install MySQL
3. Import project into IDE (NetBeans / Eclipse / IntelliJ)
4. Configure DB credentials in DBConnection.java
5. Run Main/java.java
