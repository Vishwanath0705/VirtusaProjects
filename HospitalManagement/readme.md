# Hospital Management System (SQL Analytics Project)

## Overview
This project is a Hospital Management Database system built using MySQL. It focuses on database design, data insertion, and analytical queries to generate insights such as doctor performance, revenue, and disease patterns.

The project contains:
- Database schema creation
- Sample data insertion
- Analytical SQL queries for reporting

---

## Project Structure


hospital_management_db/ |

|-- schema.sql -> Database structure (tables + relationships)

|-- data.sql -> Sample data insertion (patients, doctors, appointments, treatments)

|-- analysis.sql -> Analytical queries for reports and insights

|-- HospitalManagement.png -> Shows the ER diagram of the project 

---

## Database Setup (schema.sql)

This file creates the database and required tables.

### Tables:
